# 🏠 Housing Price Analysis

## Overview

This project focuses on exploring and analyzing a housing dataset to understand the factors that influence house prices. Through data cleaning and exploratory data analysis (EDA), various relationships between housing features and price are examined.

The analysis aims to identify key attributes that contribute to property value and prepare the dataset for future machine learning models.

---

## Dataset

The dataset contains information about residential properties, including:

- Price
- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road Access
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status

## Analysis Performed

### 1. Data Loading
- Imported the dataset into a Pandas DataFrame.
- Reviewed the first few records.

### 2. Dataset Exploration
- Checked dataset dimensions.
- Examined column names and data types.

### 3. Missing Value Analysis
- Verified the dataset for null values.
- Confirmed data quality before analysis.

### 4. Statistical Summary
- Generated descriptive statistics for numerical features.
- Analyzed mean, minimum, maximum, and distribution values.

### 5. Price Distribution Analysis
- Visualized house price distribution using histograms.
- Identified the overall spread of property prices.

### 6. Correlation Analysis
- Converted categorical variables into numerical form.
- Generated a correlation heatmap to understand feature relationships.

### 7. Area vs Price Analysis
- Examined the relationship between house area and price.
- Observed a positive correlation between property size and value.

### 8. Bedrooms vs Price Analysis
- Compared house prices across different bedroom counts.
- Analyzed how bedroom numbers impact pricing.

---

## Key Findings

- House area has a strong positive impact on price.
- Properties with more bedrooms and bathrooms generally cost more.
- Air conditioning and preferred locations contribute to higher prices.
- Furnished houses tend to have higher market value.
- The dataset is clean and suitable for predictive modeling.
